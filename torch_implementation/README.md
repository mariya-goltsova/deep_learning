# ДЗ1

Домашнее задание 1. Полносвязные нейронные сети. Бэкпроп.

В этом задании вам предстоит реализовать свой фреймворк для обучения нейронных сетей на основе numpy. Интерфейс фреймворка будет максимально копировать PyTorch, так что вы немного познакомитесь с тем, как все устроено изнутри. Директория modules содержит файлы с шаблонами фреймровка, а tests — тесты для проверки корректности ваших реализаций.
